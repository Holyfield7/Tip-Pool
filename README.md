# 💸 Tip-Pool

Tip-Pool is a Web3-powered tipping and micro-payment platform that enables instant, transparent, and fair peer-to-peer tipping using blockchain wallets.

---

## 🚀 Vision

Traditional tipping systems are centralized, slow, opaque, and expensive. Creators, service providers, and teams often face delayed payouts, unfair distribution, and lack of transparency.

Tip-Pool solves this by providing:
- Instant peer-to-peer tipping
- Transparent wallet balances
- Pooled tips with fair distribution
- Low-cost micro-transactions
- Trustless, blockchain-based settlement

---

## 🧠 Why Tip-Pool?

✔ No intermediaries  
✔ Real-time wallet balances  
✔ Fair tip pooling  
✔ Web3-native payments  
✔ Scalable for creators, teams, and communities  

---

## 🏗️ Architecture Overview

- **Backend:** PHP (REST APIs)
- **Database:** MySQL
- **Blockchain:** ERC-20 compatible tokens
- **Wallet Logic:** Balance tracking, credit, debit, and tipping
- **API Testing:** Postman

---

## 🔑 Core Features

- User creation
- Wallet balance initialization
- Wallet credit & debit
- Peer-to-peer tipping
- Tip pooling logic
- Transparent transaction flow

---

## 🧪 API Endpoints (Example)

| Method | Endpoint        | Description              |
|------|-----------------|--------------------------|
| POST | /users          | Create a user            |
| GET  | /wallet/balance | Fetch wallet balance     |
| POST | /wallet/credit  | Credit wallet            |
| POST | /wallet/tip     | Send a tip               |
